== Description ==

This is a simple contact form.


== Installation ==

1. Upload `foie-contact.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[foie-contact]` in your templates
1. Add an 'ENC_KEY' constant of 32 characters to encrypt the password used for PHPMailer SMTP Mail Transfer
